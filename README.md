# 🍽️ Smart WhatsApp Restaurant Agent

## 🚀 Project Introduction
Excited to share my latest project – **Smart WhatsApp Restaurant Agent** 🤖🍽️  

This is an AI-powered WhatsApp chatbot that helps restaurants automate customer interactions and improve service efficiency.

---

## ✨ Key Features
- 📋 View menu instantly  
- 🛒 Place orders via WhatsApp  
- 📍 Get restaurant details  
- 💬 Real-time automated responses  
- ⚡ Fast and user-friendly experience  

---

## 💡 Project Idea
This project demonstrates how **AI + automation** can transform small businesses by:
- Saving time  
- Reducing manual work  
- Enhancing customer experience  

---

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- WhatsApp API  
- AI Integration
